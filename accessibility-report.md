I used the **Chrome Lighthouse** audit tool while running the site on a local server.

## Improvements & Findings
- **Image Alt Text:** Added descriptive text to my portrait so screen readers can describe it.

- **Contrast:** Lighthouse noted a minor contrast issue, which I will look into for future versions to improve readability.
